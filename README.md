# 问道长生

> 一念成路，一世成书。

《问道长生》是一款以文字叙事、人物关系和轮回为核心的修仙角色扮演游戏。

你会从一名初入道途的修士开始，在宗门旧案、家族遗债、宿敌邀战与六个时代的变迁中作出选择。救下谁、相信谁、欠下什么、又在何处退让，都会进入角色状态与命途长卷；有些结果当场显现，有些要到数章之后，甚至下一世才会回来。

## 开始游戏

### Windows 玩家

1. 从[最新 Windows 构建](https://github.com/Iukulete/wendao-changsheng/actions/workflows/godot-windows.yml)中下载名为 `wendao-changsheng-*-windows-x86_64` 的压缩包。
2. 完整解压到一个可写文件夹，不要直接在压缩包内运行。
3. 双击 `wendao-changsheng.exe`。

游戏支持 Windows 10/11 64 位系统，建议分辨率不低于 `1280 x 720`。游玩不需要联网，也不需要安装额外运行库。

## 这一世会经历什么

- **读完属于自己的长篇。** 主线以卷章推进，正文会记住前面的行动、承诺、债务和人物立场，不把选择压缩成一串数值加减。
- **在看得见与看不见的后果之间判断。** 选项只描述眼前行动；风险藏在人物态度、环境细节和已有线索里，需要你自己权衡。
- **只在因果追上来时迎战。** 战斗由剧情中的敌踪触发，开战前会说明争夺之物；回合中可以读取敌方意图、组织三拍破势，也可以承担代价撤离。
- **走过完整的一生。** 修炼、突破、寿元和死亡彼此相连。轮回只会在一世终结后按因果触发，前世留下的定局仍会影响后来者。
- **探索可选的镜湖秘境。** 你的道途、装备、关系和前世记忆会化为能力牌；路线、压力、心魔与首领阶段共同构成一段独立但会回到主线的历练。
- **看见世界继续向前。** 势力、人物和时代不会因为玩家离场而静止。旧友会成长，旧债会过期，未处理的敌意也可能在多年后反噬。

## 操作

所有主要操作都可以用鼠标完成，也可以使用键盘：

| 场景 | 键位 |
| --- | --- |
| 主界面 | `1` 开始新生，`C` 继续，`I` 导入旧档，`O` 音频设置 |
| 游历界面 | `I` 行囊，`A` 玉兵，`L` 命途长卷，`S` 手动封存，`O` 音频设置，`Esc` 返回菜单 |
| 剧情选择 | 数字键选择行动，`Esc` 暂离当前事件 |
| 战斗 | `1`-`3` 使用战技，`4` 服药，`5` 或 `Esc` 撤离，`L` 查看战斗记录 |
| 镜湖秘境 | 数字键选择路线或能力，`E` / `Space` 结束回合，`Esc` 离开秘境 |

界面底部会根据当前场景显示可用操作，不需要记住全部快捷键。

## 存档与轮回

存档位于游戏文件夹内的 `save/`，便于备份和迁移。章节结果、战斗结算和退出流程会自动封存，也可以随时按 `S` 手动保存。主档损坏时，游戏会尝试恢复上一份完整备份，并保留损坏文件以供排查。

旧 Win32 版的 `SAVE_V4` / `SAVE_V5` 六槽存档可以从主菜单只读导入。把原有 `slot_1.txt` 至 `slot_6.txt` 放在游戏同级的 `save/` 目录即可；导入不会修改原文件。

## 内容提示

剧情会触及死亡、战争、强制控制、囚禁、创伤与不对等权力关系等沉重主题。相关路线以人物处境和后果为重点，不把强迫关系包装成亲密奖励。含蓄的成年亲密关系仅建立在明确自愿的前提上。
