# 《问道长生》Windows 版

> 一念成路，一世成书。

《问道长生》是一款以文字叙事、人物关系和轮回为核心的修仙角色扮演游戏。你在这一世留下的承诺、债务、敌意和情分，可能当场改变局面，也可能在多年后或下一世重新出现。

## 开始游玩

1. 完整解压游戏文件夹，不要直接在压缩包内运行。
2. 将文件夹放在桌面、游戏目录或其他可写位置。
3. 双击 `wendao-changsheng.exe`。

游戏支持 Windows 10/11 64 位系统，建议分辨率不低于 `1280 x 720`。游玩不需要联网，也不需要安装额外运行库。

## 操作

所有主要操作都可以用鼠标完成，也可以使用键盘：

| 场景 | 键位 |
| --- | --- |
| 主界面 | `1` 开始新生，`C` 继续，`I` 导入旧档，`O` 音频设置 |
| 游历界面 | `I` 行囊，`A` 玉兵，`L` 命途长卷，`S` 手动封存，`O` 音频设置，`Esc` 返回菜单 |
| 剧情选择 | 数字键选择行动，`Esc` 暂离当前事件 |
| 战斗 | `1`-`3` 使用战技，`4` 服药，`5` 或 `Esc` 撤离，`L` 查看战斗记录 |
| 镜湖秘境 | 数字键选择路线或能力，`E` / `Space` 结束回合，`Esc` 离开秘境 |

界面底部会根据当前场景显示可用操作，不需要记住全部快捷键。

## 存档

存档位于游戏文件夹内的 `save/`。章节结果、战斗结算和退出流程会自动封存，也可以按 `S` 手动保存。

请在复制或移动游戏时一并保留 `save/`。主档损坏时，游戏会尝试恢复上一份完整备份，并保留损坏文件以供排查。

旧 Win32 版的 `SAVE_V4` / `SAVE_V5` 六槽存档可以从主菜单只读导入。将原有 `slot_1.txt` 至 `slot_6.txt` 放入游戏同级的 `save/` 目录；导入不会修改原文件。

## 常见问题

**双击后没有启动**

确认游戏已经完整解压，并且 `wendao-changsheng.exe` 与 `wendao-changsheng.pck` 位于同一目录。不要从压缩包、只读目录或受限的系统目录中直接运行。

**没有声音**

先在游戏的音频设置中确认总音量与静音状态，再检查 Windows 当前输出设备。切换输出设备后，重新启动游戏可让音频后端重新连接。

**无法写入存档**

把整个游戏文件夹移动到当前用户可写的位置，例如桌面或专用游戏目录，并确认安全软件没有阻止游戏创建 `save/`。

## 内容提示

剧情会触及死亡、战争、强制控制、囚禁、创伤与不对等权力关系等沉重主题。含蓄的成年亲密关系仅建立在明确自愿的前提上。
