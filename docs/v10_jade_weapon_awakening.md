# v0.10 轮回玉兵觉醒与显圣

## 目标

v0.9 已让成就奖励成为跨轮回永久玉兵。v0.10 继续解决“玉兵只是固定属性装备”的问题，使当前共鸣武器能随真实游戏行为成长、觉醒，并主动释放道法。

## 共鸣成长

当前共鸣玉兵会从以下行为获得共鸣：

- 日常修炼与闭关
- 历练事件
- 战斗胜利
- 层级提升
- 境界突破

每件武器独立保存：

- 共鸣值
- 觉醒阶段
- 显圣蓄能
- 显圣次数

切换武器不会把原武器的成长转移给新武器。

## 三层觉醒

| 共鸣 | 阶段 | 作用 |
|---:|---|---|
| 0～29 | 沉眠 | 使用基础成就奖励属性 |
| 30～119 | 初鸣 | 属性增强，并开始提供路线加成 |
| 120～299 | 真名 | 进一步增强属性和路线效果 |
| 300+ | 道化 | 达到当前玉兵的完整道法形态 |

武器首次进入新阶段时，会复用三档成就弹窗系统显示觉醒通知，并进入弹窗队列，不覆盖正在展示的成就。

## 四类玉兵道法

### 杀伐

代表：斩劫古刀、血煞断魄刀、九霄名器、通天残星枪。

- 觉醒后更偏向攻击提升。
- 提高历练事件成功修正。
- 显圣时大量增加修为，并获得少量灵石。

### 护生

代表：青莲护生杖、岁月长生木剑、祖庭镇世剑、轮回断界刀。

- 觉醒后更偏向防御与最大气血。
- 提供稳定的破境辅助。
- 显圣时恢复大量气血，高阶段可凝成丹药。

### 问心

代表：青霄问心剑、承道玉简剑、百劫照尘锋、无垢心锋。

- 觉醒后更偏向道心。
- 提供较强破境辅助和适度事件修正。
- 显圣时增加修为与道心。

### 万道

代表：万道归一刃、黑白轮回剑、四象巡道戟、无量玉皇兵。

- 攻、防、气血、道心均衡增长。
- 对破境和历练都提供较高修正。
- 显圣时同时补充修为、气血与道心。

## 操作

- `[A]`：查看成就、藏兵、共鸣、阶段与蓄能。
- `[Y]`：切换下一件已解锁玉兵。
- `[J]`：蓄能达到 100 后令当前玉兵显圣。

显圣会消耗当前武器全部蓄能，但不会降低共鸣或觉醒阶段。

## 数值边界

- 同一时间仍只允许一件玉兵与角色共鸣。
- 觉醒按基础属性百分比增强，并附加少量流派属性，不会让全部永久武器同时叠加。
- 切换武器时先撤销旧武器有效加成，再施加新武器有效加成。
- 读档时只恢复已经写入角色数值的当前有效加成，不重复累加。

## 存档兼容

- 成就存档升级为 `ACHIEVEMENTS_V3`。
- 兼容 `ACHIEVEMENTS_V1` 与 `ACHIEVEMENTS_V2`。
- V1/V2 旧武器会以共鸣 0、沉眠阶段开始，但保留解锁状态与当前装备。
