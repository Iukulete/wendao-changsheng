# 产品体验二期平衡遥测报告

日期：2026-07-18

范围：玩法数值、选择取舍、遭遇风险和剧情节奏；不包含美术制作或审计。

## 样本设计

平衡探针复用 `ten_thousand_playtest_test.gd` 的真实规则路径，并由
`run_playtest_10k.ps1` 汇总分片遥测。角色五行改为实际的 `1..10` 随机分布，
每十局固定加入一局全 `1` 低根和一局全 `10` 高根，以覆盖构筑边界。

本轮调参样本为连续 600 局、四分片、种子索引 `0..599`。六种模式均匀覆盖
事件、剧情、修炼、突破、普通战斗、秘境、存档、轮回和本地 AI 降级。

## 选择取舍

旧目录允许同一事件的三个选择全部没有显式代价，且野心、造化从未出现负向
取舍。修正后门禁要求每个事件或剧情选择组最多保留一个无代价选项，并按
`enmity +N`、其他玩家字段负值或道途负值识别代价。

```text
随机事件：36 个，108 个选择，13 个无代价
连续剧情：8 组，24 个选择，8 个无代价
六道覆盖：
ambition +32/-3, bonds +40/-12, compassion +36/-17,
creation +32/-3, defiance +36/-4, insight +41/-2
600 局实际展示的无代价选项比例：22.83%
```

目录验证和运行时剧情验证都会拒绝重新引入整组三个或两个纯收益选择。

## 遭遇风险

普通战斗的敌人旧缩放每升一层只增加 `3%`，远低于玩家固定升层属性，600 局
在所有根值组均为 100% 胜率。修正为首层 `100%`、每升一层增加 `14%` 后：

```text
普通战斗：293 胜 / 7 负，胜率 97.67%
低根组：38 胜 / 7 负，胜率 84.44%
标准组：232 胜 / 0 负
高根组：23 胜 / 0 负
平均回合：5.347
平均气血损失：21.75%
濒危战斗：12 / 300，4.00%
```

强根仍提供可靠优势，标准角色需要承担可感知消耗，低根角色则必须考虑装备、
丹药或避战，不再能无风险刷取战斗奖励。

秘境保持独立风险曲线：286 次完成、7 次失败，完成率 97.61%；其中低根组
14 次完成、6 次失败，完成率 70.00%，标准和高根组保持接近全通。

## 剧情节奏

连续剧情门禁记录真实触发事件序号：

```text
今生主线：16 幕，事件 0..45，最小/最大间隔均为 3
二世续章：12 幕，事件 0..33，最小/最大间隔均为 3
```

每幕之间固定留出两次普通抉择，完整主线和续章都能在单世合理长度内收束。
