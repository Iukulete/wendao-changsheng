# v0.9 成就弹窗与轮回玉藏兵

## 目标

把成就从静态列表升级成即时反馈和永久养成系统：

- 成就达成时，从窗口底部缓慢上浮并渐显；停留后向下渐隐。
- 成就分为三档：灵玉、玄金、天命。
- 三档使用不同边框、颜色、粒子数量、辉光和停留时间。
- 每项成就奖励一件永久玉兵。
- 玉兵保存在黑白轮回玉中，不属于当世普通器物，不受纪元断代、天地侵扰或轮回散失。

## 动画

- 灵玉：青色单框、轻粒子、约 4.8 秒。
- 玄金：紫金双框、中等粒子、约 5.6 秒。
- 天命：赤金多重辉光、冠纹、强粒子与脉冲，约 6.5 秒。
- 动画采用约 520ms 的缓出上浮，停留后约 850~1050ms 下沉渐隐。
- 多个成就同时达成时进入队列，按顺序播放，不互相覆盖。

- 万道归一刃
- 轮回断界刀
- 承道玉简剑
- 百劫照尘锋
- 无垢心锋
- 九霄名器
- 四象巡道戟
- 黑白轮回剑
- 通天残星枪
- 无量玉皇兵

新武器解锁时会自动共鸣评分更高的玉兵。游戏中按 `[Y]` 循环切换已解锁玉兵；按 `[A]` 查看成就和完整藏兵列表。

玉兵提供攻击、防御、气血和道心加成。新一世创建时自动重新显化；保存和读取时不会重复叠加属性。
