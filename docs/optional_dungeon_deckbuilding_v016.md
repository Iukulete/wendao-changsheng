# v0.16 可选秘境构筑玩法边界

## 定位

卡组构筑不是《问道长生》的主线玩法，也不替代修炼、人物关系、动态世界、跨世剧情或普通战斗。

它只在镜湖秘境、纪元遗迹、心魔劫等明确的副本入口中启用。玩家离开副本后立即回到原有修仙主循环。

## 牌的含义

牌不是脱离角色的外部收藏物。每张牌都是角色已有能力在副本规则中的临时投影：

- 基础攻击和防御形成通用招式。
- 六大道途形成各自的签名灵诀。
- 当前共鸣玉兵形成对应流派的玉兵道法。
- 装备、器痕、人物关系与前世记忆可继续提供副本能力。
- 创伤和心魔形成需要主动处理的心障牌。

角色永久拥有的是功法、道途、玉兵、器痕和记忆。副本只负责把这些能力临时组成牌组并改变抽取顺序，不建立与角色成长平行的永久卡牌库存。

## 已实现基础

- `DungeonSystem` 独立状态域，不改变普通 `CombatSystem`。
- 镜湖空阙秘境四层路线：交锋、机缘、休整、炼器、强敌与首领。
- 14 张基础/道途/玉兵/心障灵诀。
- 5 张起手、3 点灵力、抽牌/弃牌/消耗、护体、虚弱、成长和恢复。
- 公开敌方意图与独立的秘境气血。
- 心魔压力满值后生成心障残页，压力回落但牌组被污染。
- 确定性路线、洗牌、敌人和自动运行硬上限。
- 副本完成、失败或撤离都有明确结算，并写入有界历史。
- 副本失败不会直接替代主线死亡；它留下气血代价与“秘境心创”。
- 副本完成只带出修为、灵石、材料和玉兵共鸣。
- 进行中副本可进入版本化存档；轮回会清除未完成副本并保留历史。

## 当前交互

- 主界面提供明确的“镜湖空阙秘境”入口。
- 路线、战斗、撤离、失败与通关均有独立页面和结算。
- 续档会直接恢复进行中的秘境路线或战斗。
- 数字键、鼠标与回合结束操作均已接入，不改变普通战斗快捷键。

## 后续扩展

1. 把角色功法、NPC 羁绊和器痕进一步映射为能力来源，不增加永久卡包抽取。
2. 为不同纪元补充专属路线节点、敌人意图、心障和首领规则。
3. 增强牌面图像、施法反馈、敌方受击和压力临界演出。
4. 保持副本为可选内容，不把主界面历练自动改成卡牌战斗。
